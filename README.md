# tryhackme-labs
This respository documents Cybersecurity labs completed on TryHackMe.
## Completed Rooms
### Introduction to Cyber Security
This room introduces cybersecurity fundamentals and explores different career paths within the cybersecurity industry.
Topics covered:
-Overview of cybersecurity
-Importance of protecting systems
-Cybersecurity career roles such as security analyst, Penetration Tester and Digital Forensics Analyst.
### What is Networking?
This room introduces the fundamentals of computing networking
Topics covered:
-Identifying devices on a network
-Internet definition
-Ping(ICMP)
### DNS in detail
This room introduces how DNS works and how it helps one access internet services.
Topics covered:
-Domain Hierarchy
-Record Types
-Making a request
